# AmbitionBox Company Scraper

Scrapes the top 50 companies from AmbitionBox and extracts structured workplace rating data.

## 🔗 Live Report
👉 [View Interactive Dashboard](https://nikhilsarwl.github.io/ambitionbox-scraper/companies_report.html)

## Files
- `scraper.py` — Python scraper (requests + BeautifulSoup)
- `companies_formatted.xlsx` — Output data (50 companies, 12 columns)
- `companies_report.html` — Interactive web dashboard

## How to run
```bash
pip install requests beautifulsoup4 lxml
python scraper.py
```# ambitionbox-scraper
